# biblio_composants

Monorepo pour une bibliothèque de composants React réutilisables et une application Next.js de démonstration.

## Structure

- `/packages/components` : Librairie React (TypeScript)
- `/apps/demo` : Application Next.js pour la démo et la documentation

## Démarrage rapide

1. Installer les dépendances :
   ```bash
   npm install
   ```
2. Lancer l’application de démo :
   ```bash
   cd apps/demo
   npm run dev
   ```
3. Développer tes composants dans `packages/components` et les importer dans l’app de démo.

## Scripts utiles
- `npm run build` : Build de la librairie
- `npm run dev` : Démarrage du serveur de démo
